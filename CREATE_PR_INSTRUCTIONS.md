# Pull Request Creation Instructions

## 🚀 Create PR via GitHub Web Interface

### Step 1: Navigate to GitHub Repository
**URL:** https://github.com/Great-2025/Grant-Stream-Contracts

### Step 2: Create Pull Request
1. Click on the "Pull requests" tab
2. Click the "New pull request" button
3. Select the following:
   - **Base repository**: `Great-2025/Grant-Stream-Contracts`
   - **Base branch**: `main`
   - **Head repository**: `Great-2025/Grant-Stream-Contracts`
   - **Compare branch**: `feature/multi-regional-tax-jurisdiction-mapping`

### Step 3: Fill PR Details
**Title:** `Implement Multi-Regional Tax Jurisdiction Mapping Logic - Issue #207`

**Description:** Copy the contents from `PR_TAX_JURISDICTION_DESCRIPTION.md`

### Step 4: Add Labels and Assignees
- **Labels:** `finance`, `compliance`, `legal`
- **Assignees:** Add appropriate team members
- **Reviewers:** Request reviews from core team

### Step 5: Create PR
Click the "Create pull request" button to submit.
